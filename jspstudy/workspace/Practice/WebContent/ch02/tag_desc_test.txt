<ch2. 스크립트태그>

	1.  스크립트 태그의 종류
		1) 선언문    <%! %>
		  : 자바 변수나 메소드의 정의!(호출x)
		  : 스크립틀릿 태그보다 나중에 선언해도 스크립틀릿 태그에서 사용 가능 
		  : 서블릿 프로그램으로 번역될 때 클래스 수준의 멤버 변수가 되므로 전역변수로 사용
		2) 스크립틀릿 <%  %>  
		  : 일반 자바 로직, out내장객체를 사용하지 않아도 HTML응답 만들기 가능 
		  : 매소드 내부에 복사되므로 지역변수가 되어 이 태그에 선언된 변수는 스크립틀릿 태그 내에서만 사용 가능 
		3) 표현문    <%=  %>
		  : 변수,계산식, 메소드 호출 결과를 문자열 형태로 출력 
		  : 웹 브라우저에 출력할 부분을 표현

  	   스크릿틀릿은  지역변수, 변수 선언가능
